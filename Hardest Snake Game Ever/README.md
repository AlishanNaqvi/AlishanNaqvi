# Snake Game

This is a unique Snake Game built using Python's Turtle graphics. 
The game features dynamic speed, score multipliers, and a unique color pattern for the snake.

## Features
- Dynamic snake colors that change as the snake grows.
- Increasing difficulty with a score multiplier.
- Bonus food items that occasionally appear for extra points.

## How to Play
1. Run the game by downloading the file or cloning it.
2. Use the **`W`, `A`, `S`, and `D` ** keys to move the snake.
3. Eat the food to grow the snake and increase your score.
4. Avoid hitting the **walls** or **yourself**!
